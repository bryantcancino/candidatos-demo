# candidatos-demo
Creacion de modulo de candidatos

Esta API simula el modulo de gestion de candidatos. Se entrega con el proxy, el contrato Swagger econ OpenApi 2 y 3 (solo fue probado en la 2); y la coleccion de consumo. Para consumirlo son necesarios tres  elemento:

1. Username
2. Password
3. base_path

Dichos elementos no se presentan, por motivos de seguridad (Nunca se suben credenciales).
